# 校园文化解码 — 交互式校园文化地图网站

**访问链接**：https://neuqcampusmap.netlify.app/

---

## 项目简介

本网站是东北大学秦皇岛分校"校园文化解码"劳动实践项目的核心成果，定位为一款交互式校园文化地图。用户通过点击校园地图上的地标热区，即可了解该地标的历史、功能与人文故事，并支持打卡互动与路线导航。

---

## 核心功能

1. **地标点击浏览** — 点击地图地标，弹出名称、简短摘要、详细介绍及实景图片。
2. **打卡互动系统** — 支持地标打卡，打卡状态通过 `localStorage` 本地存储，地图对应位置显示绿色"✓"徽章，支持一键清空打卡记录。
3. **路线展示与按路线打卡** — 内置多条文化路线（"新生入门路线""校史文化与人文路线""校园自然与摄影路线"），选择路线后站点自动弹窗，地标弹窗中显示"下一站：......"选项，支持直接跳转查看或打卡。
4. **跨端适配** — 兼容电脑端与移动端，优化触屏操作，静态数据加载，无需登录，访问流畅且保护用户隐私。

---

## 技术实现

- **地图热区**：使用 Image Map Generator 标注地图热区，嵌入 `index.html`。
- **前端交互**：原生 JavaScript 实现点击事件处理、弹窗展示、打卡状态管理与路线导航逻辑。
- **数据存储**：打卡状态基于 `localStorage` 持久化；地标数据以 `data.json` 格式结构化存储（17个地标，34篇文案，42张图片）。
- **移动端适配**：优化触屏 `touch` 事件，处理与 `click` 事件的兼容与防抖动。
- **部署上线**：通过 Netlify 平台连接 Git 仓库，实现自动构建与发布。

---

## 数据资源

`data.json` 包含 17 个校园文化地标的标准化资料，每个地标含简短介绍与详细文化介绍两版文案，以及高清实景图片路径。数据结构清晰，可在其他应用场景中复用。
